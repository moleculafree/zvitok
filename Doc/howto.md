Якщо натискання кнопки Видалити організацію якимось чином не видаляє файл профілю, то:

> [!IMPORTANT]  
> Видалення файлу видаляє усю інформацію по організації (відправлені звіти, ключі що додавались в програму).


## Госпаде! В програмі вірус/троян!
По перше: проєкт ZvitOkPlus не вірус, не троян.  
По друге: часто Defender Windows (антивірус) видаляв файл програми у зв'язку з підозрою на вірус.  Це помилка роботи антивірусу (false positive).  
Це може відбуватись:
- через відсутність повноцінного цифрового підпису ехе-файлу;
- є функціі звантаження/шифрування (робота з сертифікатами);
- компілятор (програма що формує кінцевий готовий exe файл) доволі старий, але зручний для створення будь-яких маленьких корисних програм.  
Все це в сукупності може призводити до помилкового визначення загрози антивірусом (false positive).

Проблему можна вирішити, **додавши файл до списку виключень антивірусу**. Якщо такий підхід для вас неприйнятний — утримайтесь від використання програми.


## Ключі від monobank не працюють! (спойлер: працюють)
При спробі додати ключ Підпису від monobank програма залишає порожні поля Назва, Код ДРФО, ЕДПОУ/ДРФО.  
Відбувається це тому, що сертифікат від monobank містить неповні дані, якщо порівнювати з "єталоним" сертифікатом від приватбанку.
Для розв'язання цієї проблеми в Налаштуваннях програми встановіть галочку "Додавати відсутні дані в профіль з INI файлу" [дізнатись більше](/Doc/README.md) 


## Обраний ключ не є електронним підписом
Оновіть програму до [актуальної версії](https://github.com/moleculafree/zvitok/releases/latest)


## Сертифікат не знайдено

По перше:
Оновіть файли сертифікатів ДПС (вручну):
1. Відкрийте сторінку [на сайті Державної податкової служби](https://tax.gov.ua/elektronna-zvitnist/platnikam-podatkiv-pro/edina-adresa)
2. Знайдіть на сторінці два файли з розширенням .cer:
* сертифікат для підписання повідомлень ДПС (квитанції, інформаційні повідомлення)
* сертифікат для шифрування повідомлень до ДПС (звітність тощо)
3. Завантажте ці файли до папки програми \Certificates\


По друге:  
Оновіть файли сертифікатів ЦЗО вручну:
1. Завантажити файл кореневих сертифікатів з сайту [https://czo.gov.ua/certificates](https://czo.gov.ua/download/certificates/CACertificates.p7b)
2. Даний файл треба розмістити в папку програми (не розпаковуємо) - \Certificates\CACertificates.p7b

> [!NOTE]
> Для оновлення файлів ЦЗО можна скористатись [автоматичним завантаженням оновлень сертифікатів](https://github.com/moleculafree/zvitok/blob/main/Doc/README.md#%D0%B7%D0%B0%D0%B2%D0%B0%D0%BD%D1%82%D0%B0%D0%B6%D1%83%D0%B2%D0%B0%D1%82%D0%B8-%D0%BE%D0%BD%D0%BE%D0%B2%D0%BB%D0%B5%D0%BD%D1%96-%D1%84%D0%B0%D0%B9%D0%BB%D0%B8-%D0%B1%D0%B5%D0%B7-%D0%BF%D0%B5%D1%80%D0%B5%D1%85%D0%BE%D0%B4%D1%83-%D0%BD%D0%B0-%D1%81%D0%B0%D0%B9%D1%82)


## В списку відсутній КНЕДП
> [!NOTE]
> Скористайтесь [автоматичним завантаженням оновлень сертифікатів](https://github.com/moleculafree/zvitok/blob/main/Doc/README.md#%D0%B7%D0%B0%D0%B2%D0%B0%D0%BD%D1%82%D0%B0%D0%B6%D1%83%D0%B2%D0%B0%D1%82%D0%B8-%D0%BE%D0%BD%D0%BE%D0%B2%D0%BB%D0%B5%D0%BD%D1%96-%D1%84%D0%B0%D0%B9%D0%BB%D0%B8-%D0%B1%D0%B5%D0%B7-%D0%BF%D0%B5%D1%80%D0%B5%D1%85%D0%BE%D0%B4%D1%83-%D0%BD%D0%B0-%D1%81%D0%B0%D0%B9%D1%82)

1. Перевірити та встановити оновлення програми. Якщо після оновлення потрібний КНЕДП не з'явився в переліку перейти до наступного пункту.
2. Скачати актуальні налаштування сумісних КНЕДП (ЦСК) з сайту [https://czo.gov.ua/en/certificates](https://czo.gov.ua/download/certificates/CAs.json)  
Даний файл розмістити в папку програми - \Certificates\CAs.json  
Перезапустити програму.
3. Якщо краще не стало - пишіть [issues](https://github.com/moleculafree/zvitok/issues) або на [форум](https://opz.org.ua/forum/viewforum.php?f=39)

## ZvitOk/ZvitOkPlus не запускається , при запуску грузит процесор на 100%
**Все виснет лагает минут 5-10 и потом ничего тишина приложение не открывается, вы там что майните чтоли чтото)) в чем может быть проблема не могу отчеты отправить!

Відкрити Провідник, на файлі ZvitOkPlus.exe права кнопка, Властивості, закладка Сумісність.  
В рамці "Режим сумісності" вибираємо Віндовс 8. Внизу кнопка "Змінити налаштування для всіх користувачів".  
В рамці "Режим сумісності" вибираємо Віндовс 8. Застосувати, Ок.  
Вверху вікна кнопка "Запустити засіб усунення неполадок повязаних з несумісністю." А далі все по вищеописаній інструкції.  
Повинно запрацювати.  
**Дійсно робочий працюючий варіант - це запускати програму "Від Адміністратора"

Ще один варіант вирішення проблеми з фрізами: права кнопка миші - Властивості - Безпека - далі надати "Повний доступ" для програми.
